# תרגיל לולאות

בתרגיל זה תלמד להשתמש בלולאות JavaScript כדי לבנות תכנית אמיתית.

---

## חלק א׳ — בחירת אמוג׳י (emojis.js)

השלם את הפונקציה `fillEmojiPicker()` כך שתמלא את טבלת בחירת האמוג׳ים.

הפונקציה צריכה לעבור על כל קודי Unicode בין `0x231A` ל-`0x1FAFA`
ולהוסיף לטבלה כל אמוג׳י צבעוני.

**פונקציות שכבר כתובנו בשבילך:**
- `isColorfulEmoji(char)` — בודקת האם תו הוא אמוג׳י צבעוני
- `addEmojiToPicker(char)` — מוסיפה אמוג׳י לטבלה
