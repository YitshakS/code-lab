# תרגיל בלולאות מקוננות

## הקדמה:

פרויקט זה מדמה את העולם האמיתי, בו לרוב הדרישה ממך תהיה לעבוד על קוד קיים שלא נכתב על ידיך, ולא להתחיל מסקראץ'. האם זה אומר שעליך ללמוד ולהכיר את כל הקוד? ממש לא!
הפרויקט יסגל לך כישורים לאתר את הקוד המינימלי הנדרש להבנה לצורך ביצוע המשימה שלך ולדלג על הרוב המוחלט שלא רלוונטי.
בפרויקט יהיה עליך להוסיף פיצ'רים מגניבים.
בכל שלב באפשרותך להיעזר ב ai על ידי לחיצה על 💡. הוא רואה את הקוד שלך ומסוגל לתת לך רמזים, תיקונים, Code Review וכל שמה שיסייע לך להתקדם.
ה ai מונחה להיות המורה הפרטי שלך ולא לגלות לך את הפתרון, אבל כמובן באפשרותך לבחור לצפות בפתרון ע"י לחיצה על 👁.

<br/>

בצד השמאלי של המסך נמצא הקוד, ובצד הימני הפלט.
לפני המשך קריאת התרגיל מומלץ לך לחקור בצד הימני כיצד אמור להיראות הפתרון.
בתפריט יש לבחור את הצורה והמאפיינים המרכבים אותה ובהתאם לכך היא תוצג.
בקוד התרגיל תינתן דוגמה אחת לאמוג'י ודוגמה אחת לצורה. עליך יהיה להשלים את אפשרות הבחירה של כל האימוג'ים וכל הצורות.
מעל הקוד מופיעים הטאבים הבאים:

<br/>

* README.md - קוד Markdown לצפיה בלבד 🔒, המכיל את ההוראות בהן הינך צופה ממש ברגע זה. לחיצה על 📝 מציגה את הקוד ולחיצה על 📖 מציגה את הפלט שלו.
* index.html - קוד HTML לצפיה בלבד 🔒, המציג את האלמנטים של התפריט והצורות.
* style.css - קוד CSS לצפיה בלבד 🔒, המציג את הסגנונות של האלמנטים, לדוגמה גודל כל אמוג'י.
* script.js - קוד JavaScript לצפיה בלבד 🔒, המכיל את הלוגיקה, לדוגמה יצירת.
* emojis.js - קוד JavaScript אותו עליך להשלים, המכניס את האמוג'ים הצבעוניים לחלונית בחירת האמוג'י. לחיצה על ✏ מעבירה למצב עריכה בו ניתן להזין את הפתרון ולחיצה על 👁 מגלה את הפתרון.
* shapes.js - קוד JavaScript אותו עליך להשלים, המגדיר כל צורה נבחרת בהתאם למאפיינים המוגדרים בתפריט. לחיצה על ✏ מעבירה למצב עריכה בו ניתן להזין את הפתרון ולחיצה על 👁 מגלה את הפתרון.

## לולאות מקוננות:

"לולאות מקוננות" הן לולאה בתוך לולאה. למעשה אין בהן אף חידוש והן לולאות רגילות לכל דבר.
לולאה מורכבת מאיטרציות. כל איטרציה מבצעת את כל משימותיה ורק בסיומה מתבצע מעבר לאיטרציה הבאה, וכן הלאה, עד לסיומה של הלולאה.
אם המשימה כוללת פקודה, בכל איטרציה הפקודה מופעלת מחדש.
באופן זהה, אם נחליף את הפקודה בלולאה, נקבל בדיוק את אותו הדבר:
אם לולאה מכילה בתוכה לולאה, בכל איטרציה של הלולאה החיצונית, הלולאה הפנימית מופעלת מחדש, מבצעת שוב את כל האיטרציות שלה ורק בסיומן מתבצע מעבר לאיטרציה הבאה של הלולאה החיצונית.

### דוגמה - לולאה מקוונת להצגת לוח הכפל:

הלולאה החיצונית עוברת על העמודות והפנימית על השורות.

```js
for (let i = 1; i <= 10; i++) {
  let row = '';
  for (let j = 1; j <= 10; j++)
    row += `${i * j}\t`;
  console.log(row);
}
```

## חלק א׳ — בחירת אמוג׳י (emojis.js)

השלם את הפונקציה `fillEmojiPicker()` כך שתמלא את טבלת בחירת האמוג׳ים.

הפונקציה צריכה לעבור על כל קודי Unicode בין `0x231A` ל-`0x1FAFA`
ולהוסיף לטבלה כל אמוג׳י צבעוני.

**פונקציות שכבר כתובנו בשבילך:**
- `isColorfulEmoji(char)` — בודקת האם תו הוא אמוג׳י צבעוני
- `addEmojiToPicker(char)` — מוסיפה אמוג׳י לטבלה
