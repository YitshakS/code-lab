# זאב, עז וכרוב

חידת המעבורת הקלאסית — ממומשת ב-CSS בלבד, ללא שורת JavaScript אחת.

## הבעיה

חקלאי צריך להעביר זאב, עז וכרוב לגדה הימנית באמצעות סירה קטנה.  
הסירה יכולה לשאת את החקלאי ויצור אחד בלבד בכל פעם.

**האילוצים:**
- זאב ועז לא יכולים להישאר לבד — הזאב יאכל את העז
- עז וכרוב לא יכולים להישאר לבד — העז תאכל את הכרוב

## הפתרון הטכני

הפאזל ממומש כ-**state machine ב-CSS טהור**:

- כל מצב אפשרי של המשחק מיוצג על ידי `input[type="radio"]`
- לחיצה על ישות בוחרת את ה-radio המתאים ומציגה את הלוח הבא
- אנימציות מעבר מבוצעות עם `@keyframes` ו-`transition`
- הסלקטור `input:checked ~ .wrapper` מאפשר שליטה מלאה על ה-UI ללא JS

## נסה

לחץ על ישות בגדה השמאלית כדי להעלות אותה לסירה ולחצות את הנהר.
